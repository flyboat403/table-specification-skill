# table-specification 技能

## 职业教育考试命题双向细目表编制工具

> **将考试大纲转化为结构化的命题蓝图，让命题工作从"无图纸施工"变为"精准施工"**

---

## 📖 项目简介

双向细目表（Test Item Specification Table / Exam Blueprint）是命题工作的"施工图纸"——它将抽象的课程标准转化为具体的评估工具，规划题型、分值、难度分布，确保考点覆盖全面、难度分布合理、能力层级匹配。

本技能为 AI Agent 提供专业的双向细目表编制能力，支持：

- **三种课程类型**：认知类、动作技能类、情感态度类
- **三套目标分类体系**：布卢姆认知分类、辛普森动作技能分类、克拉斯沃尔情感分类
- **完整工作流程**：从大纲解析到评分指南编制的一站式服务
- **输出格式多样化**：Markdown、Excel、Word

---

## ✨ 核心功能

### 1. 课程类型智能识别

根据课程名称和内容特征，自动判断课程类型并推荐对应的目标分类体系：

| 课程类型 | 识别关键词 | 推荐分类体系 |
|----------|------------|--------------|
| **认知类** | 理论、原理、管理、营销、会计、计算机 | 布卢姆认知分类 |
| **动作技能类** | 体育、运动、实训、操作、技能、烹饪 | 辛普森动作技能分类 |
| **情感态度类** | 德育、思政、心理、职业素养 | 克拉斯沃尔情感分类 |

### 2. 大纲智能解析

支持多种输入格式：
- PDF/Word 文档（自动提取文本）
- 直接粘贴大纲内容
- 口头描述大纲要点

自动提取核心要素：科目名称、考试时长、总分值、题型构成、模块划分、知识点列表、目标层级要求。

### 3. 三维分析框架

构建"知识点-目标层级-应用情境"三维矩阵，确保：
- **知识点维度**：大纲考点全覆盖
- **目标层级维度**：层级匹配大纲要求
- **情境维度**：真实职业工作场景嵌入

### 4. 细目表自动编制

生成结构化双向细目表，包含：
- 序号、知识点、目标层级、题型、题号
- 单题分值、预估难度、命题要点/情境提示
- 主观题采分点分配

### 5. 自动校验与平衡优化

核对约束条件：
- ✓ 总分值匹配大纲规定
- ✓ 各题型题数与分值匹配
- ✓ 各模块分值占比合理
- ✓ 难度比例分布合理（低30%、中50%、高20%，可调整）
- ✓ 目标层级分布符合大纲要求

### 6. 主观题评分指南

为所有主观题生成完整评分指南：
- 采分点分配（精确分值）
- 满分标准
- 扣分情形

## 🚀 使用方法

### 触发场景

当用户提及以下关键词时，本技能会被激活：

- **中文关键词**：试卷命题、考试设计、题型分配、考点分布、分值规划、命题蓝图、考试大纲、命题细目表、双向细目表
- **英文关键词**：test blueprint、exam specification、assessment design、test item specification、vocational education exam

即使只是笼统地说"帮我出一份试卷"或"设计一套考试题"，也会触发本技能。

### 基本使用流程

```
用户: 请根据以下《机械加工技术》考试大纲编制双向细目表
      [大纲内容...]

Agent: [Step 0] 课程类型识别 → 认知类 → 布卢姆认知分类
       [Step 1] 大纲解析 → 提取核心要素
       [Step 2] 三维分析框架构建
       [Step 3] 双向细目表编制
       [Step 4] 校验与平衡优化
       [Step 5] 主观题评分指南
       → 输出完整双向细目表
```

### 输入格式示例

```markdown
【考试科目】机械加工技术
【考试时长】90分钟
【总分值】100分

【题型构成】
- 单选题：15题，每题2分，共30分
- 填空题：5题，每题4分，共20分
- 简答题：2题，每题10分，共20分
- 综合题：1题，共20分

【考核模块与知识点】
模块一：切削基础（建议占比15%）
- 切削运动分类（识记）
- 切削用量三要素（理解）
- ...
```

## 🎯 支持的目标分类体系

### 布卢姆认知分类（认知类课程）

| 层级 | 定义 | 典型考核行为 |
|------|------|--------------|
| 识记 | 记忆和回忆知识 | 定义、列举、背诵、识别 |
| 理解 | 把握知识含义 | 解释、说明、归纳、比较 |
| 应用 | 在新情境中使用知识 | 计算、操作、实施、运用 |
| 分析 | 分解材料理解结构 | 区分、分析、比较、组织 |
| 评价 | 基于标准做出判断 | 评判、评价、论证、选择 |
| 创造 | 产生新的观点或产品 | 设计、建构、规划、创作 |

### 辛普森动作技能分类（动作技能类课程）

| 层级 | 定义 | 典型考核行为 |
|------|------|--------------|
| L1 知觉 | 感知动作情境 | 观察、识别、辨别动作特征 |
| L2 定向 | 准备活动、心理准备 | 准备、调整、热身活动 |
| L3 有指导反应 | 在指导下模仿动作 | 模仿、跟随、重复基本动作 |
| L4 机械化 | 动作熟练化 | 熟练完成标准动作 |
| L5 复杂外显反应 | 复杂动作组合 | 技术串联、技战术运用 |
| L6 适应 | 适应新情境 | 情境应对、战术调整 |
| L7 创新 | 创造新动作模式 | 编排、设计、创造新动作组合 |

### 克拉斯沃尔情感分类（情感态度类课程）

| 层级 | 定义 | 典型考核行为 |
|------|------|--------------|
| 接受 | 对刺激的感知和接受 | 注意、倾听、接受 |
| 反应 | 主动参与和回应 | 参与、回应、讨论 |
| 价值评价 | 认同某种价值 | 肯定、支持、认同 |
| 组织 | 整合不同价值 | 比较、整合、形成信念 |
| 价值内化 | 价值观成为行为准则 | 实践、坚持、内化为行为 |

## ⚠️ NEVER 反模式

本技能严格禁止以下行为：

- **NEVER 对体育/技能类课程使用布卢姆认知分类**——会导致效度失效
- **NEVER 脱离大纲添加考点**——会造成考核效度问题
- **NEVER 给应用级题目配无情境的纯理论题**——职业教育考核岗位就绪能力
- **NEVER 忽略推断性设计声明**——必须声明设计决策依据
- **NEVER 让综合题采分点模糊不清**——采分点必须可测量、可量化
